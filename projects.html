Memory Card Game

I believe memory card games is a fun and essential way to increase one's own memory, so it's important to start young, so this project was made with a young-end user like a child in mind. I used vibrant and attracting colors to appeal for younger audience, I used a gradient background to blend the two colors. I believe with a good design with these kinds of colors need a fun and funky type of font style, so I used a 3rd party font-style. The card game itself is triggered by a start button, and I used JavaScript to add the dynamics of the game itself. One of the features I integrated with this project was a counter to keep score of the tries it takes for a person to complete the game.

I am a front-end Developer with design skills to bring any creative idea to the web.
