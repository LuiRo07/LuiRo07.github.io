Memory Card Game

I think memory card games are a fun way to improve one's own memory, so it's important to start young. This project was made with young end users as the target audience. I used vibrant and attracting colors and a blend of funky colors for a gradient background. To pair the colors well, I used a playful font styling using a 3rd party font. The card game itself is triggered by a start button. I used JavaScript to add the dynamics of the game itself. One of the features I integrated with this project was a counter to keep score of the tries it takes the user to complete the game.
